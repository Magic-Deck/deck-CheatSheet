# Ponyshow Cheat Sheet

Learn the Ponyshow syntax using this cheat sheet deck.

## [View Online](http://ponyshow.github.com/cheatsheet)

## Install and run

Download this deck, install `ponyshow` command and run...

```
npm install ponyshow -g
pony run
```
